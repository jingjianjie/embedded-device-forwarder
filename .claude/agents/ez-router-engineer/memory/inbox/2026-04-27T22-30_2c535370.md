# RPD 阶段 0 任务 0.1 修复 + tests/unit 占位

- **任务**: 修 plugin handler 返回值越界(契约 5)。reactor.c:259 调用 handler 后无边界校验,返回 >1024 会越过栈缓冲 buf[1024] 与 event_msg_t.data[MAX_DATA] 双重边界。
- **产出**:
  - `routerd/src/reactor.c:258-272`: handler 返回值 clamp 到 [0, MAX_DATA],`<0 → drop+continue`,`>MAX_DATA → truncate+WARN`
  - `routerd/include/plugin_loader.h:6-11`: 在 `plugin_handler_t` typedef 上方加契约 5 注释(只写 why)

- **新发现的契约/约束**:
